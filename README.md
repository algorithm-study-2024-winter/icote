## 정리 규칙

1. 각자 이름으로 폴더 생성해서 작성하기 (main branch 하나로 사용할 것이기 때문에 절대 절대 남의 폴더 건드리지 않기 !)
2. 주차별로 폴더 생성해서 관리하기
3. 주차별 폴더 안에는 문제 출처 구분하여 (boj, book, programmers 등)
4. 각 문제 파일의 가장 상단에 해당 문제 링크 기재하기 (책 제외)
5. 코드에는 항상 주석을 잘 달기
6. 개인정보를 업로드하지 않도록 주의하기
